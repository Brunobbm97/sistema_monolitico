Sistema monolítico usando Python/Flask, Postgresql como banco de dados
Na pasta conectores esta presente arquivos.py que conectam a aplicação com o banco de dados
Na pasta classes esta os métodos construtores
Na pasta controladores está os métodos que controlam as rotas da aplicação, podendo realizar CRUD em cada classe

Comandos necessários: 
python -m venv venv
inicializar a venv
pip install requirements.txt
flask run
